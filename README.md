# Cardiac Disease Prediction System

👨‍💻 Developed by: Suraj Mulik & Santosh Hajare

---

## 📌 Project Description

This project predicts whether a person is at risk of cardiac (heart) disease using machine learning techniques. The prediction is based on medical attributes such as age, cholesterol level, blood pressure, and other health parameters.

The system uses trained machine learning models and provides predictions through an interactive web application built using Streamlit.

---

## 🎯 Objective

* To analyze heart disease data
* To build a predictive machine learning model
* To assist in early detection of cardiac diseases

---

## 📊 Dataset

The dataset used in this project is taken from Kaggle and contains various medical attributes of patients.

Dataset Link:
https://www.kaggle.com/datasets/johnsmith88/heart-disease-dataset

---

## ⚙️ Technologies Used

* Python
* Machine Learning (Scikit-learn)
* Pandas, NumPy
* Matplotlib, Seaborn
* Streamlit

---

## 🧠 Machine Learning Models Used

* Logistic Regression
* Decision Tree
* Random Forest

---

## 🌐 Web Application

This project includes a Streamlit web application where users can:

* Enter medical details (age, blood pressure, cholesterol, etc.)
* Get prediction (Heart Disease: Yes/No)
* View results instantly

---

## ▶️ How to Run the Project

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```
